                   (2025년 최신 버전)

🚀 빠른 시작 가이드

1️⃣ 첫 번째: 프로그램 설치
   👆 install.bat 파일을 【우클릭】 → 【관리자 권한으로 실행】
   ⏳ 자동으로 설치가 진행됩니다 (2-3분 소요)
   ❌ Node.js 오류가 나면: https://nodejs.org/ 에서 설치 후 재실행

2️⃣ 두 번째: 프로그램 실행
   🚀 개발용: run_dev.bat 더블클릭 (권장)
   🏢 운영용: run_prod.bat 더블클릭

3️⃣ 세 번째: 웹사이트 접속
   🌐 5초 후 브라우저가 자동으로 열립니다
   📍 또는 수동으로 http://localhost:5000 접속

4️⃣ 네 번째: 관리자 로그인
   🔐 우상단 "관리자" 버튼 클릭
   🔑 비밀번호: 0000 입력

5️⃣ 다섯 번째: 프로그램 종료
   🛑 실행 창에서 Ctrl+C 누르거나 창 닫기

=================================================================
📁 파일 설명

install.bat     → 🔧 프로그램 설치 (관리자 권한 필요)
run_dev.bat     → 🚀 개발/테스트용 실행 (권장)
run_prod.bat    → 🏢 실제 운영용 실행
사용방법.txt    → 📖 이 파일 (사용법 설명)

=================================================================
✨ 새로운 기능

🔹 자동 브라우저 실행: 서버 시작 5초 후 자동으로 웹페이지 열림
🔹 포트 충돌 감지: 이미 사용 중인 포트 자동 감지
🔹 관리자 권한 확인: 설치 시 권한 자동 확인
🔹 상세한 진행 상황: 각 단계별 상태 표시
🔹 오류 자동 진단: 문제 발생 시 해결 방법 제시

=================================================================
❓ 문제 해결

🔴 설치 안됨
   → Node.js 미설치: https://nodejs.org/ 에서 LTS 버전 설치
   → 권한 부족: 우클릭 → "관리자 권한으로 실행"
   → 인터넷 연결: 안정적인 네트워크 확인

🔴 실행 안됨
   → install.bat 먼저 실행했는지 확인
   → 포트 충돌: 다른 프로그램 종료 후 재시도

🔴 웹사이트 안열림
   → 서버 시작 메시지 확인 (5-10초 대기)
   → 방화벽 설정 확인
   → 백신 프로그램 실시간 검사 일시 해제

🔴 관리자 로그인 안됨
   → 비밀번호: 0000 (숫자 제로 4개)
   → 브라우저 쿠키/캐시 삭제 후 재시도

=================================================================
💡 사용 팁

✅ 권장 사용 순서
   install.bat (최초 1회) → run_dev.bat (매번 사용)

✅ 개발 vs 프로덕션
   • 개발 모드: 코드 수정 시 자동 재시작 (개발/테스트용)
   • 프로덕션 모드: 최적화된 실행 (실제 서비스용)

✅ 자주 사용하는 기능
   • 이벤트 생성: 관리자 → 대시보드 → "새 이벤트"
   • 파일 제출: 메인 페이지 → 이벤트 선택 → 파일 업로드
   • 제출 확인: 관리자 → 대시보드 → 이벤트 목록

✅ 네트워크 공유
   다른 컴퓨터에서 접속하려면 서버 컴퓨터의 IP 주소로 접속
   예: http://192.168.1.100:5000
